# 📘 Angular User Authentication Application  

---

## 📌 Project Overview  
This project demonstrates a basic **User Authentication System using Angular**. It includes functionality for user registration, login, and displaying user data on a profile page.  

The application is built using Angular components, routing, and services, and is intended for practical learning purposes.

---

## 🚀 Features  

- ✔️ User Registration  
- ✔️ User Login  
- ✔️ User Profile Display  
- ✔️ Routing between pages  
- ✔️ Local storage-based authentication  

---

## ⚙️ Setup & Installation  

### 🔹 Step 1 – Install Node.js  
Download and install Node.js from the official website:  
``` https://nodejs.org/  ```

Verify installation:  

```
node -v
npm -v
```

---

### 🔹 Step 2 – Install Angular CLI  

```
npm install -g @angular/cli
ng version
```

---

### 🔹 Step 3 – Create Angular Project  

```
ng new user-auth-app
cd user-auth-app
```

Select:  
- Routing: Yes  
- Stylesheet: CSS  

---

### 🔹 Step 5 – Generate Components and Service  

```
ng generate component register
ng generate component login
ng generate component profile
ng generate service auth
ng generate service user 
```

These commands automatically create required folders and files.

---

### 🔹 Step 4 – Run Application  

```
ng serve
```

Open in browser:  
```
http://localhost:4200/
```

---

### 🔹 Step 6 – Add Project Code  

> 📦 **Note:**  
> All required code for this project (components, service, routing, and logic) is already included in the uploaded repository files.  
> You only need to use or replace files as provided.

## ⚠️ Important Notes  

- This project does not use a backend server  
- Data is handled using browser local storage  
- SSR/SSG is not required  
- Ensure all repository files are correctly placed before running
